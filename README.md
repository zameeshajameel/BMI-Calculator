# BMI Calculator App (Jetpack Compose)

A simple Android application built using Kotlin and Jetpack Compose to calculate Body Mass Index (BMI) based on user input of height and weight.

## Features

* Calculate BMI based on height and weight
* Shows BMI category (Underweight, Normal, Overweight, Obese)
* Clean and modern Jetpack Compose UI
* Input validation for safe calculations
* Reset option to clear inputs

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3

## How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on an emulator or physical device

## Project Structure

* `BMIScreen.kt` → Main UI screen
* `InputCard.kt` → Reusable input component
* `ResultCard.kt` → Displays BMI result
* `ui.theme` → App colors and theme
